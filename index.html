Publication / Research

Attendance System Using Face Identification | (IJRAR)

    In recent years, face recognition-based attendance systems have gained attraction, but many of users still face key limitations: high hardware costs, poor real-time performance, internet dependency, limited features, and privacy concerns. Our project offers a low-cost, efficient alternative using a standard webcam and Python libraries like `face_recognition and OpenCV. It ensures fast, real-time operation with minimal internet use, auto-generates absentee lists, sorts attendance by roll number, and supports multi-session tracking. Optional features like WhatsApp alerts and Google Drive backups further enhance usability, making this system ideal for budget conscious educational institutions.
